# Frá námi til starfs – samhengi og markmið

- **Hvati:** Brúa bilið milli fræðilegs náms og vinnulífs með verkferlum og tólum sem eru *
  *raunverulega notuð** í fyrirtækjum.
- **Kjarni:** Gagnsæ samvinna, rekjanleiki, endurtakanleiki, ábyrgð og fagmennska.
- **Aðferð:** Sameina **agílt vinnulag**, **GitHub/GitHub Classroom**, **pull requests** og *
  *jafningjarýni** sem **kennslu- og matsinnviði**.
- **Ávinningur fyrir nemendur:** Sýnilegt framlag, skýr hlutverk, traust og undirbúningur fyrir
  atvinnulíf.
- **Ávinningur fyrir kennara:** Betri yfirsýn, réttlátara mat, skilvirkara lokaúrvinnsluferli.

**Verkfæri sem mynda innviði kennslunnar**

- Git, GitHub, Issues, Branches, Pull Requests
- GitHub Classroom sem verkefna- og samvinnuumgjörð
- Skýr vinnuferli og „handover“ beint í geymslu
