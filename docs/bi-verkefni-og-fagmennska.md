# Iðnaðarverkefni og fagmennska í BI

**Markmið:** Hækka atvinnulegt notagildi og fagmennsku með **raunheimum**:

- **Sama gagnasafn** allt misserið (minni uppsetningarálag, dýpri skilningur)
- **Samstarf við fyrirtæki** (gögn, samhengi, viðmiðin trausts)
- **Capstone** sem samþættir og ítrar fyrri lotur
- **TBL/PBL** með iRAT/tRAT/tAPP/iREF og framvirka ígrundun

## Mat og endurgjöf sem þróunarferli

- Sama **gæðaviðmið** yfir lotur → endurgjöf safnast upp, ekki sundrast
- **Sýnileg verkfæri** (PR, útgáfustýring) gera framlag einstaklings metanlegt
- **Mismunandi áhorfendur**: akademía leyfir „þróun“ → atvinnulíf krefst **trúverðugleika**

## Helstu lærdómar

- Greinileg **aukning í þátttöku** og fagmennsku
- Skýrari **væntingar** milli kennara, nemenda og fyrirtækis
- Kennsla á Git/GitHub **færð fyrr** í námsbraut til að minnka byrði í flóknu verkefni
